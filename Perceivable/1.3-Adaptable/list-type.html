List Type Demo

Usage in HTML:

There are 3 list types used for different purposes. Unordered Lists (<ul>): Used for items that don't require a specific order, typically marked with bullet points. Ordered Lists (<ol>): Used for items that have a specific sequence, typically numbered. Description Lists (>dl>): Used for pairs of terms and descriptions, consisting of <dt> for the term and <dd> for the description Indexing of each list should start from the first element in the list and not from the type of the list. Example: 

                <ul>
                    <li> 
                    <li>
                </ul>
