Table Identification Demo

Usage for Table coded with <table> tag in HTML:

The <table> element is used to define a table in HTML, which is useful for displaying tabular data such as lists, schedules, or statistics. Within a table, the <tr> tag is used to define rows, the <th> tag is used to define header cells (typically bold and centered), and the <td> tag is used for standard data cells. 

              
                <table>
                  <thead>
                    <tr>
                      <th>First Name</th>
                      <th>Last Name</th>
                      <th>Address</th>
                    </tr>
                  </thead>
                  <tbody>
                    <tr>
                      <td>Fred</td>
                      <td>Jackson</td>
                      <td>123 Broad St.</td>
                    </tr>
                    <tr>
                      <td>Sara</td>
                      <td>James</td>
                      <td>123 Broad St.</td>
                    </tr>
                  </tbody>
                </table>

Usage of Table coded with role="table" in HTML:

The <table> element is used to define a table in HTML, which is useful for displaying tabular data such as lists, schedules, or statistics. With the addition of role="table", this explicitly defines the element as a table, making it more accessible to assistive technologies. Within a table, the <tr> tag, with role="row", is used to define rows. The <th> tag, with role="columnheader", defines header cells, which are typically bold and centered to represent column headings. The <td> tag, with role="cell", is used to define standard data cells within each row. 

              
                <div role="table" style="display: table;>
                  <div role="row" style="display: table-row;">
                    <div role="columnheader" style="display: table-cell;">First Name</div>
                    <div role="columnheader" style="display: table-cell;">Last Name</div>
                    <div role="columnheader" style="display: table-cell;">Address</div>
                  </div>
                  <div role="row" style="display: table-row;">
                    <div role="cell" style="display: table-cell;">Fred</div>
                    <div role="cell" style="display: table-cell;">Jackson</div>
                    <div role="cell" style="display: table-cell;">123 Broad St.</div>
                  </div>
                  <div role="row" style="display: table-row;">
                    <div role="cell" style="display: table-cell;">Sara</div>
                    <div role="cell" style="display: table-cell;">James</div>
                    <div role="cell" style="display: table-cell;">123 Broad St.</div>
                  </div>
                </div>
